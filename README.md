# AWS-Fun-Facts-App
“I built a serverless full-stack application using AWS where a frontend hosted on Amplify calls an API Gateway endpoint backed by Lambda, which retrieves data from DynamoDB.”


# ☁️ Cloud Fun Facts App

🚀 Live Demo: https://production.dvyom28xvxs0r.amplifyapp.com

---

## 📌 Project Overview
This is a serverless full-stack application built using AWS.

Users can click a button to generate cloud computing facts powered by:
- AWS API Gateway
- AWS Lambda
- Amazon DynamoDB
- AWS Amplify (frontend hosting)

---

## 🧠 Architecture

Frontend (Amplify) → API Gateway → Lambda → DynamoDB

---

## ⚙️ Tech Stack

- HTML, CSS, JavaScript
- AWS Lambda
- API Gateway
- DynamoDB
- Amplify

---

## 🎯 Features

- Generate random cloud computing facts
- Fully serverless architecture
- Scalable backend with AWS services
